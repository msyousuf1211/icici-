# ICICI Bank Product Portfolio Website

A comprehensive product information website to help understand and present ICICI Bank's range of financial products to customers.

## Features

- **Savings Accounts**: Regular, Premium, and Family savings account options
- **Loan Products**: Home, Car, Personal, Education, Business, and Gold loans
- **Credit Cards**: Platinum, Travel, Shopping, and Premium cards
- **Investment Products**: Fixed Deposits, Mutual Funds, Recurring Deposits, and Portfolio Management
- **Insurance Products**: Life, Health, Motor, and Home insurance
- **Digital Services**: Mobile banking, Internet banking, and UPI services

## How to Use

1. Simply open `index.html` in any modern web browser
2. Navigate through different product categories using the header menu
3. Each product card contains:
   - Key features and benefits
   - Target customer segment
   - Interest rates (where applicable)
